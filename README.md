# 📚Library Management System

[cite_start]A production-ready, secure, and responsive web application built to fulfill the requirements of the Zetech University [cite: 2, 9] [cite_start]ICT Diploma project[cite: 38]. [cite_start]This project aligns with the official Project Supervision Form [cite: 7][cite_start], tracking development from planning and UI design to database integration and deployment[cite: 16, 18].

## ✨ Features

* **Secure Authentication:** Admin login system featuring bcrypt password hashing and session fixation prevention.
* **Catalog Management:** Add new books with unique IDs/ISBNs and view the current inventory with real-time availability status.
* [cite_start]**Student Registration:** Register students using their official Admission Numbers (ADMNO) [cite: 5] and course details.
* **Circulation Desk:** A streamlined interface for issuing and returning books.
* **Activity Tracking:** A comprehensive activity log that records every borrowing and returning action, including timestamps and student details.
* **Responsive UI:** A clean, modern interface built with Bootstrap 5, fully functional on both desktop and mobile devices.

## 🛠️ Technologies Used

* **Backend:** PHP (Object-Oriented Programming)
* **Database:** MySQL (using PDO for secure, prepared statements)
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Architecture:** Modular, multi-page application (MPA)
